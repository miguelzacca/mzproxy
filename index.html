🚀 mzproxy

Proxy universal para liberar qualquer requisição CORS. Suporta GET, POST, PUT, DELETE mantendo headers, body e binários.

Como Usar

GET /api/proxy/https://api.exemplo.com/dados
POST /api/proxy/https://api.exemplo.com/enviar

Alternativa Seguro para URLs longas: Envie o cabeçalho x-target-url.

fetch('/api/proxy', {
  headers: {
    'x-target-url': 'https://api.exemplo.com/dados'
  }
})


🎭 Web Scraper Anti-Bot

Utilize o endpoint /api/proxy/scrape para baixar sites com renderização completa (JavaScript), contornando bloqueios avançados (Cloudflare, Akamai etc.) utilizando Chromium Headless, Stealth Plugins e simulação de cursor fantasma.

GET /api/proxy/scrape/https://www.site-protegido.com/
// Exemplo rápido:
const res = await fetch('/api/proxy/scrape/' + encodeURIComponent('https://www.zoom.com.br'));
const htmlCompleto = await res.text();
